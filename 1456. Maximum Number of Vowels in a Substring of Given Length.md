# 251203

Sliding Window 的經典題型，用暴力解也不難。

# 題目

給定一個字串 s 和一個整數 k。求所有長度為 k 的子字串 (substring) 中，包含最多母音 (vowels, 英文 a/e/i/o/u) 的情況，回傳最大母音數量。

## Sliding Window

用一個長度為 k 的視窗 (window)，代表目前考慮的子字串，並動態更新。
* 起始視窗是 `s[0..k-1]`，統計其中母音數量 cur。
* 把視窗「向右滑」一格：
  * 移除最左邊一個字元 (如果是母音 → cur -= 1)
  * 加入右邊新進的一個字元 (如果是母音 → cur += 1)
* 每次滑動後更新最大值 maxcount = max(maxcount, cur)

這樣最終 maxcount 即為答案。

* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$

## 暴力法

* 時間複雜度: $O(n * k)$
  * 循環找每個子字串: $O(n-k)$。
  * 遍歷子字串內的母音: $O(k)$。
* 空間複雜度: $O(1)$
