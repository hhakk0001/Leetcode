# 題目

輸入一整數陣列 nums，還有一個 linked list 的首節點 head。
刪除所有值存在於 nums 中的節點，並回傳修改後的首節點。

## 第一次嘗試: 雜湊表(集合)

將 nums 裝入 set，依序走訪串列並檢查節點是否需刪除。

* 時間複雜度: $O(n+m)$，n 是 nums 大小，m 是串列大小
* 空間複雜度: $O(n)$
