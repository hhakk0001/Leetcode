# 題目

給定一個正整數陣列 `nums` 以及一個整數 `k`，請回傳連續子陣列（subarray）中，其所有元素的乘積嚴格小於 `k` 的子陣列數量。

```
Input: nums = [10,5,2,6], k = 100
Output: 8
```

## 滑動視窗

使用左右兩指標，列舉右邊界並維護左邊界，  
若當前視窗內的元素乘積大於 k，則移動左邊界向右，直到乘積小於 `k`。

因為元素皆為正整數，右邊界不變的情況下移動左邊界只會使乘積變小，  
代表在找到第一組符合條件的子陣列後，繼續右移左邊界產生的視窗乘積也小於 `k`。

> 承上例: `nums = [10,5,2,6], k = 100`。  
> 列舉到 `2` 時，可能的子陣列有 `[5, 2], [2]`。

也就是說，若左右邊界的指標為 `left, right`，  
則可知以 right 為右邊界的子陣列個數為 `right - left + 1` 個。

* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$
