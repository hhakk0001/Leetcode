# 題目

給定一個字串 `s`，請找出不包含重複字元的最長子字串（substring），並回傳其長度。

注意: 子字串必須是**連續的**。

## 滑動窗口

用滑動窗口維護子字串，雜湊表紀錄字元的出現次數。

* 時間複雜度: $O(n)$
* 空間複雜度: $O(1)$，嚴格來說是 $O(128)$ 或 $O(len(set(s))$
