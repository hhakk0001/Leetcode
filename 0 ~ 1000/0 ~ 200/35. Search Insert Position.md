# 題目

給定一個**已排序 (嚴格遞增) 且不重複**的整數陣列 `nums`，以及一個整數 `target`。

請回傳 target 在 nums 中的位置索引，  
若 target 不存在，則回傳應該 target 應該插入的位置。


## 二分搜尋

對於一般的二分搜尋來說，當 `target` 不存在時，  
迴圈結束後，`left` 會位在「第一個大於 `target` 的元素」，  
而這個元素就是我們應該要插入 `target` 的位置。  
因此，若迴圈結束後沒找到 `target`，回傳的位置索引即為 left。

* 時間複雜度: $O(\log n)$
* 空間複雜度: $O(1)$
