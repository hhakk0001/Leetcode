# 251106

要理解圖形追蹤，小心實作邏輯。

# 題目

給定一張代表圖片的 2D 陣列 `image`，其中每個元素代表顏色值、以及一個起始座標 (sr, sc) 以及一個顏色值 newColor。
將起始像素與「所有與它有相同顏色且上下左右相連的像素」全部改成 newColor。

類似於圖像處理中的「油漆桶工具」（像 Photoshop 中填色功能）。

## 第一次嘗試: 直覺解(DFS)

照著題意做，搭配圖形追蹤的概念模擬染色的過程。

需要注意的是，在遞迴函式內檢查題目條件時，需要先檢查是否越界，不然可能觸發 IndexError。

### 修改

代表方向的 tuple 集合可以挪到遞迴函式外層，避免重複建立。

若 m 跟 n 分別是 image 的長跟寬:
* 時間複雜度: $O(m * n)$，最壞情況下會訪問所有元素
* 空間複雜度: $O(m * n)$，呼叫遞迴的深度
