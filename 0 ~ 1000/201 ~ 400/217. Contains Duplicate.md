# 251016

* 使用Set: 最佳
    * 時間複雜度: $O(n)$
    * 空間複雜度: $O(n)$
* 排序法:
    * 時間複雜度: $O(nlogn)$
    * 空間複雜度: $O(1) ~ O(n)$
* 暴力法: 雙迴圈
    * 時間複雜度: $O(n^2)$
    * 空間複雜度: $O(1)$

簡單。

## 第一次嘗試

利用 Python 中資料結構 Set 內元素不重複的特性，將數字存入集合的同時檢查該數字是否已存在於集合中。

## 排序法

將 nums 排序後線性檢查兩兩相鄰的元素。

## 暴力法

雙迴圈循環檢查每個元素。
