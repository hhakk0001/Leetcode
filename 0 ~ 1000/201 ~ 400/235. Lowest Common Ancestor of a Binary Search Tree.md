# 251203

LCA 的經典題目之一，與 [236.](236.%20Lowest%20Common%20Ancestor%20of%20a%20Binary%20Tree.md) 大同小異。

# 題目

給定一棵**二元搜尋樹**以及其中的兩個節點 p 和 q，要求找出它們的最近共同祖先（Lowest Common Ancestor, LCA）。
最近共同祖先定義為：在樹中同時擁有 p 和 q 為後代（子孫）的最深節點。注意，一個節點也可以是它自己的祖先。

注意: p and q will exist in the BST.

## 第一次嘗試

與 236. 相比，差別在於:
1. 題目保證 p、q 必存在，代表根節點必不為空，因此不需判斷空節點(其他情況都是回傳當前節點或是遞迴結果)。
2. 基於 BST 的特性，可用節點值的大小直接判斷 p、q 左或右子樹，不需等待左右子樹之遞迴走訪結果皆完成。


* 時間複雜度: $O(n)$
* 空間複雜度: $O(h)$，遞迴所用的 call stack，h 為樹高，最壞情況下為退化為 n。
