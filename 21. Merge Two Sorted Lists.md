* 迭代:
    * 時間複雜度: $O(n+m)$
    * 空間複雜度: $O(1)$
* 遞迴:
    * 時間複雜度: $O(n)$
    * 空間複雜度: $O(1)$

## 迭代法: 可以簡化的部分

```python
while list1 and list2:
    if list1.val > list2.val:
        ptr.next = list2
        ptr = ptr.next
        list2 = list2.next
    else:
        ptr.next = list1
        ptr = ptr.next
        list1 = list1.next
```
這裡 ptr = ptr.next 是通用操作，可以移出來:
```python
while list1 and list2:
    if list1.val > list2.val:
        ptr.next = list2
        list2 = list2.next
    else:
        ptr.next = list1
        list1 = list1.next

    ptr = ptr.next
```
